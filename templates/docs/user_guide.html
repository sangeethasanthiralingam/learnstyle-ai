Getting Started

Welcome to LearnStyle AI! This guide will help you get the most out of your personalized learning experience.

First Time? Start by creating an account and taking the learning style assessment quiz.

What is LearnStyle AI?

LearnStyle AI is an intelligent learning platform that uses machine learning to:

  • Analyze your learning preferences and patterns
  • Deliver personalized educational content
  • Provide real-time feedback and recommendations
  • Connect you with like-minded learners
  • Monitor your learning progress and optimize your experience

Account Setup

Creating Your Account

  1. Click the "Get Started Free" button on the homepage
  2. Fill in your registration form with:
    • Username (unique identifier)
    • Email address
    • Password (minimum 8 characters)
  3. Click "Register" to create your account
  4. You'll be automatically logged in and redirected to the dashboard

Profile Setup

After registration, you can enhance your profile by:

  • Adding a profile picture
  • Setting your learning goals
  • Specifying your areas of interest
  • Configuring notification preferences

Taking the Learning Style Quiz

The learning style assessment is the foundation of your personalized experience.

Quiz Overview

15 Questions

Carefully crafted questions designed to identify your learning preferences

10-15 Minutes

Quick and engaging assessment that won't take long to complete

How to Take the Quiz

  1. Navigate to the "Learning Quiz" section
  2. Read each question carefully
  3. Select the answer that best describes your preference
  4. Answer honestly - there are no right or wrong answers
  5. Complete all 15 questions
  6. Submit your responses

Understanding Your Results

After completing the quiz, you'll receive a detailed breakdown of your learning style:

  • Visual Learning % - How much you learn through visual aids
  • Auditory Learning % - How much you learn through listening
  • Kinesthetic Learning % - How much you learn through hands-on activities
Tip: You can retake the quiz anytime to update your learning profile as your preferences evolve.

User Roles & Permissions

LearnStyle AI features a comprehensive role-based access control system with three distinct user levels.

Role Hierarchy

Administrator

Full Access
Capabilities:
  • Complete system management
  • User account management
  • Content library management
  • System analytics and reporting
  • System configuration and settings
  • Data export and backup
  • All moderator and user features

Moderator

Limited Access
Capabilities:
  • Content management and review
  • User overview and basic management
  • Basic system analytics
  • Content approval and moderation
  • Report generation
  • All user features

User

Basic Access
Capabilities:
  • Personal dashboard and learning
  • Learning style assessment
  • AI tutor chat
  • Content consumption and progress tracking
  • Personal settings and preferences
  • Access to all learning features

Role Assignment

User roles are assigned by administrators and can be changed at any time. Role changes take effect immediately upon login.

Note: Your current role is displayed in the navigation menu and dashboard. Contact an administrator if you believe your role needs to be updated.

Administrator Features

Administrators have access to comprehensive system management tools and analytics.

Admin Dashboard

Access the admin dashboard through the "Admin Panel" link in the navigation menu (visible only to administrators).

Key Features:

User Management

View, edit, and manage all user accounts. Create new users, modify roles, and manage account status.

Content Management

Manage the entire content library. Add, edit, and remove educational materials.

System Analytics

View comprehensive analytics including user activity, learning style distribution, and system performance.

System Settings

Configure system parameters, manage integrations, and control feature availability.

Quick Actions

  • Export Data: Download user and analytics data for external analysis
  • System Backup: Create system backups for data protection
  • View Logs: Access system and error logs for troubleshooting
  • User Reports: Generate detailed user activity and progress reports

Moderator Features

Moderators have access to content management and user oversight tools.

Moderator Dashboard

Access the moderator dashboard through the "Moderator" link in the navigation menu (visible to moderators and administrators).

Key Features:

Content Review

Review and moderate educational content before publication. Approve or reject content submissions.

User Overview

View user information and activity. Monitor user engagement and provide support.

Basic Analytics

Access basic system statistics and user activity metrics.

Report Generation

Generate and export reports on content usage and user activity.

Content Management

Moderators can:

  • Add new educational content to the library
  • Edit existing content and metadata
  • Remove inappropriate or outdated content
  • Organize content by categories and difficulty levels
  • Set content visibility and access permissions

User Management

Moderators can:

  • View user profiles and activity
  • Toggle user account status (active/inactive)
  • View user learning progress and engagement
  • Provide user support and assistance
